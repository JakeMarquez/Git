Why does creating a character need to be so difficult?

  • Fill out the form
    Select an option by clicking inside the input field, and again on a option from the dropdown menu. Select a option for the race, class, background, and optional subrace category before moving on.
  • More Information
    Scroll down to view descriptions of available classes, races, subraces, and backgrounds.
  • Specific information
    If you would like information on a specific option, select it in the dropdown menu, and then click the categories corresponding logo.
